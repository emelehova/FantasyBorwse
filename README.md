# FH Browser (MAUI)

Кроссплатформенная версия FH Browser, созданная с помощью .NET MAUI.

## Сборка

### macOS

Сборка выполняется через GitHub Actions. Готовый `.pkg` установщик можно скачать из артефактов последнего успешного запуска.

### Windows

Сборка выполняется локально через Visual Studio.
